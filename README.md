# CRUD Generator
Crea aplicaciones web completas a partir de tu base de datos. La aplicación genera un CRUD completo a partir de las tablas seleccionadas de una base de datos pre-existente. Permite crear consultas personalizadas. La aplicación generada se puede modificar posteriormente. Ideal para crear plantillas para proyectos más avanzados.
