Requisitos de la tabla de usuarios

Para que la autenticación funcoque, tu base de datos debe contener una tabla llamada user con los siguientes campos:

  • id - Identificador único (clave primaria)
  • username - Nombre de usuario único
  • password - Contraseña hasheada con password_hash()
  • name - Nombre completo del usuario (opcional)
Asegúrate de que las contraseñas estén almacenadas usando password_hash() de PHP.
